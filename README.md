# FiredChat

 This is a full-stack real-time chat app, made using MongoDB, ExpressJS, NodeJS, ReactJS, TailwindCSS and Socket.io.
 
# ⚙️ Functionalities of Fired Chat

## 💬 Real-Time Chat System
Fired Chat provides instant real-time communication between users using Socket.IO. Messages are transmitted with very low latency, ensuring smooth and uninterrupted conversations.

### Features
- Instant messaging
- Room-based communication
- Real-time synchronization
- Low-latency message delivery

### Technologies Used
- Socket.IO
- Node.js
- Express.js

---

# 📹 Video Calling Functionality

Fired Chat supports peer-to-peer video calling using WebRTC technology. The media stream is transferred directly between users without relying on centralized media servers.

### Features
- One-to-one video calls
- Real-time audio/video streaming
- Fast connection establishment
- Reduced latency
- Browser-native communication

### Technologies Used
- WebRTC
- Socket.IO (Signaling)
- STUN/TURN Servers

---

# 🔒 Privacy-Centric Storage

Unlike traditional chat applications, Fired Chat does not store messages on centralized servers. All user chat data is stored locally inside the browser.

### Features
- Local message storage
- No centralized database
- Complete user data ownership
- Improved privacy and security
- Browser-based persistence

### Technologies Used
- IndexedDB
- LocalStorage

---

# 🤖 AI Chatbot Integration

An AI-powered chatbot is integrated directly into the chat interface to improve user engagement and assistance.

### Features
- Instant AI responses
- Explain complex terms/messages
- Context-aware assistance
- In-chat support system
